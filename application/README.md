## Lightmeter app for photography

An application that suggests settings for your manual camera based on the reading of the ambient light sensor. Can also be used in a pure lux meter mode.

## Supported sensors

- BH1750
- MAX44009

## Wiring

| Sensor | Flipper Zero |
| ------ | ------------ |
| VCC    | 3.3V         |
| GND    | GND          |
| SCL    | C0           |
| SDA    | C1           |
